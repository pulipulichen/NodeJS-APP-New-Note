# Linux

Install packages before

````
sudo apt-get install xclip build-essential -y
````

# Installation

1. `npm run docker-node-build` .
2. `npm run install-shortcut` .
3. Edit `./config/config.js`, modify the `noteFolder` .
4. Add `New Note` to the favorites.
